# Student Course Management System

## Overview

The Student Course Management System is a Java console application developed to manage student course records. The application allows users to add courses, view all courses, search for a course using its course code, compute the total number of course units, save course records to a file, and load previously saved records from a file.

This project demonstrates the use of Object-Oriented Programming (OOP) concepts and other core Java programming techniques.

---

## Features

- Add new courses
- View all recorded courses
- Search for a course by course code
- Compute total course units
- Save course records to a file (Course.dat)
- Load previously saved course records
- Prevent duplicate course codes
- Recursive search implementation
- Exception handling for invalid user input

## How to Run

1. Open the project in Visual Studio Code.
2. Ensure all Java files are in the same folder.
3. Open **Main.java**.
4. Click **Run**.
5. Follow the menu displayed in the terminal.

---

## System Menu

```
1. Add Course
2. View All Courses
3. Search Course by Code
4. Compute Total Units
5. Save to File
6. Load from File
7. Exit Program
```
